Week 4: Take-Home Mission — "The Linter's Revenge"

Mission Brief

Create a new Vite Vanilla project and experience the power of linting.

Your Task

  1. Create a new Vite Vanilla project
  2. Write a messy, poorly formatted JavaScript file with unused variables and bad indentation
  3. The Catch: Install ESLint and Prettier. Configure them. Run the linting script and take a screenshot of ESLint screaming at your bad code, then run the auto-fix command to format it perfectly

Deliverables

  • Vite project with ESLint + Prettier configured
  • Screenshot of ESLint errors before fix
  • Screenshot of clean code after auto-fix
  • Brief reflection on what the linter caught

Rubric

Criteria Points
Vite project scaffolded 20
Messy JS file created 10
ESLint installed and configured 25
Prettier installed and configured 15
Screenshots before/after 20
Reflection paragraph 10
