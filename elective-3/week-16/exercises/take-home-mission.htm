Week 16: Take-Home Mission — "The Portfolio Polish"

Mission Brief

Prepare your capstone as a portfolio-ready project for job applications.

Your Task

  1. Write a professional README.md for your capstone repository
  2. The Catch: Your README must include:

  • Project description and screenshots
  • Tech stack with badges
  • Architecture diagram
  • Setup instructions (local development)
  • API documentation (or link to Swagger)
  • CI/CD status badge
  • Live demo link

  1. Clean up your git history — squash messy commits, remove any debug code
  2. Ensure all secrets are removed from the repository

Deliverables

  • Professional README.md
  • Clean git history
  • No secrets in repository
  • All links working (live demo, API docs)

Rubric

Criteria Points
Professional README with all sections 30
Screenshots and architecture diagram 20
Clean git history 15
No secrets in repository 20
All links working 15
