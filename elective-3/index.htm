Prelim — Modern JS / TypeScript + Dev Tooling

Goal: Ramp up from basic JavaScript to modern professional TypeScript with a proper toolchain.

Week Topic Key Concepts
Week 1: Modern JavaScript Review ES6+ let/const, arrow functions, destructuring, modules
Week 2: Async JavaScript Deep-Dive Async Promises, async/await, fetch(), error handling
Week 3: TypeScript Essentials Types Interfaces, generics, any vs unknown, tooling
Week 4: Dev Tooling & Workflow Toolchain Vite, ESLint, Prettier, Git branching, PR workflow

Midterm — Component Architecture (Vanilla -> React)

Goal: Understand component thinking by building from vanilla JS to a modern framework.

Week Topic Key Concepts
Week 5: Component Thinking in Vanilla JS Components Encapsulation, props, state, composition
Week 6: React Fundamentals React basics JSX, functional components, props, useState
Week 7: Effects & Data Fetching Side effects useEffect, dependency arrays, custom hooks
Week 8: React Router & SPA Concepts SPA routing Client-side routing, lazy loading, code splitting

PreFinals — API as Product + Frontend-Backend Integration

Goal: Design, build, and consume professional-grade APIs.

Week Topic Key Concepts
Week 9: API Design Philosophy REST design Resource modeling, naming conventions, error contracts
Week 10: ASP.NET Core Web API Web API [ApiController], content negotiation, Swagger
Week 11: Frontend-Backend Integration Full-stack CORS, auth headers, token storage, useContext
Week 12: API Versioning & Documentation API lifecycle Versioning strategies, Swagger/OpenAPI, deprecation

Finals — Identity & Security + Testing/CI/CD + Capstone

Goal: Production skills — secure, test, deploy, and present a full-stack application.

Week Topic Key Concepts
Week 13: Auth Deep-Dive Auth JWT, refresh tokens, OAuth2 flow, OpenID Connect
Week 14: Testing Testing Unit (xUnit), integration (WebApplicationFactory), E2E (Playwright)
Week 15: CI/CD & Deployment DevOps GitHub Actions, build -> test -> deploy pipeline
Week 16: Capstone Final Ship Production-grade full-stack app, oral defense
