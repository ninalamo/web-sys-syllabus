Week 1: Learning Checklist (Self-Assessment)

By the end of this week, check each item you can confidently say "Yes" to:

  • [ ] I know that C# is case-sensitive and can fix capitalization errors.
  • [ ] I remember to end my statements with semicolons.
  • [ ] I can write a basic foreach loop in C#.
  • [ ] I understand how to use var properly.
  • [ ] I can create a simple Class with Properties in C#.
  • [ ] I can scaffold and run a console app with dotnet new and dotnet run.
  • [ ] I understand the build-run cycle and can read compiler errors.
  • [ ] I can use string interpolation ($"...") for formatted output.
  • [ ] I can read user input with Console.ReadLine().


How to use this checklist: Review before the next session. If you check all boxes, you're ready for Week 2.
