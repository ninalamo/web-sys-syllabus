Prelim — C# for MVC + Web Foundations

Goal: Translate VB.NET knowledge to C#, understand the web request lifecycle.

Week Topic Key Concepts
Week 1: C# Translation Bootcamp C# syntax from VB Types, properties, methods, classes, var, string interpolation, LINQ
Week 2: Interfaces & DI Mental Model Interfaces, DI Contract-based design, constructor injection, IoC container
Week 3: HTTP & The Web Request/response lifecycle HTTP methods, status codes, headers, statelessness
Week 4: MVC Big Picture MVC architecture Routing, controllers, views, models, cshtml

Midterm — ASP.NET Core MVC (The Hard Part)

Goal: Build multi-page SSR apps with routing, controllers, views, and real data flow.

Week Topic Key Concepts
Week 5: Routing Deep-Dive Routing Convention vs attribute routing, route parameters, constraints
Week 6: Controllers & Action Results Action results IActionResult, View(), Redirect(), Json(), StatusCode()
Week 7: Views & Layouts Razor views Layout pages, Partial Views, View Components, Tag Helpers
Week 8: Model Binding & Validation Forms & validation Data annotations, ModelState.IsValid, anti-forgery

PreFinals — Data & Persistence + Consuming APIs

Goal: Store and retrieve data, talk to external services.

Week Topic Key Concepts
Week 9: EF Core Fundamentals ORM DbContext, DbSet, migrations, CRUD with LINQ
Week 10: Relationships & Data Display Related data Navigation properties, eager/lazy loading, Include()
Week 11: Consuming APIs with HttpClient External APIs IHttpClientFactory, JSON serialization, error handling
Week 12: Mini-Capstone Checkpoint Integration project DB + API integration, service layer

Finals — User Interaction + Security + Capstone

Goal: Polish, secure, deploy, and present.

Week Topic Key Concepts
Week 13: Authentication & Authorization Auth Cookie auth, [Authorize], roles, Identity scaffold
Week 14: Security Fundamentals Security XSS, SQL injection, CSRF, HTTPS, OWASP
Week 15: Capstone Sprint Build sprint User stories, schema, routes, UI, auth
Week 16: Capstone Final — Deploy & Present Ship Deployment, polish, final presentations
